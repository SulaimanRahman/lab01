The purpose of this program is to print number of seconds from where the program started, then to create a child process using fork, then to wait for the child to process using waitpid, then to print the parent PID of the program, then its own PID, and then its child PID, and then print the return status of its child. After all of that it prints user time, then system timee, user tiem of child, and then system time of child. And at last it prints the number of seconds to see how long the whole process took.
